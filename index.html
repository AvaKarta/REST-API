Documentation av Det här APIet

GET / users - returnerar alla users om du är inloggad.

GET / users/{id} - returnerar användaren med samma ID om du är inloggad

POST / users - skapar en ny användere till databasen om du är inloggad.
Accepterar JSON object på formatet {"username":"unikt namn","password":""}

PUT / users/{id} - updaterar ett befintligt konto med det angivna ID om du
är inloggad. Accepterar JSON object på formatet {"username":"unikt namn","password":""}

POST / login - loggar in på ett konto. Returnerar en JWT som används vid inloggning vid det routes som kräver det.
Accepterar JSON object på formatet {"username":"","password":""}
