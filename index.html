Documentation av Det här APIet

GET / users - returnerar alla users om du är inloggad. Denna route kräver ett giltigt bearer token i authorization header.

GET / users/{id} - returnerar användaren med samma ID som är angivet. Denna route kräver ett giltigt bearer token i authorization header.

POST / users - skapar en ny användere till databasen.Denna route kräver ett giltigt bearer token i authorization header.
Accepterar JSON object på formatet {"username":"unikt namn","password":""}

PUT / users/{id} - updaterar ett befintligt konto med det angivna ID.Denna route kräver ett giltigt bearer token i authorization header.
Accepterar JSON object på formatet {"username":"unikt namn","password":""}

POST / login - loggar in på ett konto. Returnerar en JWT som används vid inloggning vid det routes som kräver det.
Accepterar JSON object på formatet {"username":"","password":""}
